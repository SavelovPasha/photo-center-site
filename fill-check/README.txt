Простой вариант сайта: только залитость по площади.

Загрузите в корень GitHub Pages все 3 файла:
- index.html
- pdf.min.mjs
- pdf.worker.min.mjs

В этом варианте убраны:
- ползунок порога;
- информация о повышенной цене;
- режимы расхода краски/цвета.

Расчёт идёт по площади: сколько процентов страницы не похоже на белый фон.
